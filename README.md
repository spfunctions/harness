# sparkco-agent

Dual Pi-Agent Runtime Harness — Phase 0

## Setup

```bash
npm install
```

## Test

```bash
npm test              # all tests
npm run test:unit     # unit tests only
npm run test:integration  # integration tests only
```

## Development

```bash
npx tsx scripts/dev.ts
```

## CLI

```bash
npx tsx src/cli/init.ts [serverUrl]
```
